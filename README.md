# atcoder-hint
## 概要
atcoderの精進中、考察が進まないが解説も見たいくない際に、解説からLLMがヒントを作成してくれるchrome拡張機能

## Get Started
### npm
```
npm install
npm run build
```
### gemini api取得
[gemini api](https://ai.google.dev/gemini-api/docs?hl=ja)

### chrome拡張機能
1. chrome拡張機能管理画面に飛ぶ
2. パッケージ化されていない拡張機能を読み込むからこのプロジェクトを読み込む
3. 拡張機能のアイコンからオプションに飛ぶ
4. gemini apiキーを登録
